## 2. Dimensiones y Especificaciones Recomendadas

### Tamaños de Imagen
- **Catálogo de productos (grid)**: 600x600px, 800x800px
- **Detalle de producto (vista grande)**: 1000x800px o 1200x900px
- **Thumbnails**: 300x300px
- **Portada/hero**: 1920x1080px (16:9)

### Formatos Aceptables
- JPG: máx. compresión sin perder calidad
- PNG: para transparencias o logos
- WebP: formato moderno, más comprimido (recomendado)

### Requisitos de Calidad
- Mínimo 800x600 píxeles
- Fondo limpio (preferido) o contexto natural
- Buena iluminación (sin sombras duras)
- Colores precisos
- Sin watermarks o logos de terceros
- Derechos de uso confirmados (CC0, comercial, etc.)

## 4. Proceso de Implementación

### Paso 1: Recopilar Imágenes
1. Crear carpeta `/public/productos/` en el proyecto
2. Subcarpetas por categoría:
   - `/public/productos/cerraduras/`
   - `/public/productos/herrajes/`
   - `/public/productos/accesorios/`
3. Nombrado estándar: `{SKU}-{ángulo}.jpg`
   - Ejemplo: `YALE-3PT-001-frontal.jpg`, `YALE-3PT-001-detalle.jpg`

### Paso 2: Optimizar Imágenes
```bash
# Usar ImageOptim, TinyPNG, o Vercel Image Optimization
# Redimensionar a tamaños estándar:
convert producto.jpg -resize 800x800 producto-small.jpg
convert producto.jpg -resize 1200x1200 producto-large.jpg
```

### Paso 3: Subir a Vercel Blob o CDN
```javascript
// Ejemplo en Next.js Server Action
import { put } from '@vercel/blob';

export async function uploadProductImage(file: File) {
  const blob = await put(
    `productos/${file.name}`,
    file,
    { access: 'public' }
  );
  return blob.url;
}
```

### Paso 4: Actualizar Base de Datos
Agregar `imagen_url` a la tabla `productos`:
```sql
ALTER TABLE productos ADD COLUMN imagen_url VARCHAR(255);

UPDATE productos
SET imagen_url = 'https://...blob.url/YALE-3PT-001-frontal.jpg'
WHERE sku = 'YALE-3PT-001';
```

### Paso 5: Mostrar en UI
```tsx
// En TarjetaProducto.tsx
import Image from 'next/image';

export function TarjetaProducto({ producto }) {
  return (
    <div className="tarjeta">
      {producto.imagen_url && (
        <Image
          src={producto.imagen_url}
          alt={producto.nombre}
          width={400}
          height={400}
          className="w-full h-48 object-cover rounded"
        />
      )}
      <h3>{producto.nombre}</h3>
      <p>${formatARS(producto.precio)}</p>
    </div>
  );
}
```
